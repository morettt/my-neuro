================================================================
  My-Neuro WebUI 控制面板 - 开箱即用包
================================================================

本压缩包提供 WebUI 图形化控制面板，统一管理 Live2D AI 助手的
各项服务（启动/停止/重启）、配置（LLM/对话/声音/云端/直播）、
日志查看、声音克隆、MCP 工具、广场资源、Live2D 动作管理等。

----------------------------------------------------------------
一、前置条件
----------------------------------------------------------------

你必须先在 my-neuro 根目录运行过 installer.py 安装 Python 环境
（会产生 my-neuro/env/python.exe）。

WebUI 大部分依赖（jinja2、click 等）已经在 env 里；env 自带未包含
的 3 个 Flask 相关包（flask、werkzeug、itsdangerous）已经随本压
缩包一起打进 live-2d/_webui_vendor/ 目录，启动 bat 会自动通过
PYTHONPATH 注入，无需任何手动安装。


----------------------------------------------------------------
二、解压步骤
----------------------------------------------------------------

将本 zip 解压到 my-neuro 根目录（与 installer.py 同级），选择
"覆盖" 即可。

正确的目标结构（解压后）：

  my-neuro/
    env/                                (installer.py 装的)
    live-2d/
      启动 WebUI 控制面板.bat            (本包提供，覆盖旧版)
      _webui_vendor/                    (本包提供：flask/werkzeug/itsdangerous)
        flask/
        werkzeug/
        itsdangerous/
      webui/                            (本包提供，覆盖旧版)
        __init__.py
        main_app.py
        ...
    README-WebUI使用说明.txt           (本文件)


----------------------------------------------------------------
三、启动
----------------------------------------------------------------

双击 my-neuro/live-2d/启动 WebUI 控制面板.bat

启动后：
  - 命令行窗口会显示 WebUI 控制面板的访问地址
  - 浏览器会自动打开 http://localhost:<自动选择的端口>
  - 关闭命令行窗口即停止 WebUI 服务

保持命令行窗口开着，不要关；窗口一关 WebUI 就停了。


----------------------------------------------------------------
四、主要功能
----------------------------------------------------------------

* 服务控制    启动/停止/重启 ASR、TTS、BERT、MemOS 等后端服务
* 配置管理    LLM 模型、API Key、对话参数、声音、云端、直播配置
* 实时日志    系统日志、桌宠日志、工具日志
* 声音克隆    上传声音模型文件、TTS 生成测试
* MCP 工具    MCP 工具列表查看
* 广场        提示词、工具、插件资源市场（在线下载）
* Live2D 管理 情绪分类管理、唱歌动作控制等


----------------------------------------------------------------
五、常见问题
----------------------------------------------------------------

Q: 双击 bat 弹出 "[错误] 未检测到 env\python.exe"
A: 你还没跑 installer.py。先去 my-neuro 根目录跑 installer.py。

Q: 浏览器没自动打开
A: 看命令行窗口里 "访问地址：http://localhost:xxxx"，手动复制到
   浏览器打开。

Q: 端口被占用
A: WebUI 启动时会自动找一个空闲端口，正常情况下不会冲突。如果
   仍然报错，关掉所有命令行窗口重新双击 bat。

Q: 想关闭 WebUI
A: 直接关掉那个命令行窗口即可（或在窗口里按 Ctrl+C）。

Q: 浏览器里显示 favicon 是 Live2D 图标
A: webui 默认引用 live-2d/fake_neuro.ico 作为图标，是从你 my-neuro
   主程序自带的图标。如果你换过 live-2d 主题，图标可能会变或消失，
   不影响功能。

Q: 改了配置后没生效
A: WebUI 的服务控制里有 "重启" 按钮，点了之后服务会重新读配置。


----------------------------------------------------------------
六、技术细节
----------------------------------------------------------------

* 框架：Flask（Python 3.11.14，env 自带）
* 端口：自动选择空闲端口（启动时显示）
* 路由：模块化蓝图（service、config、plugin、tool、marketplace、
        log、live2d 7 个蓝图）
* 国际化：支持中文/英文双语切换（i18next）
* 入口：from webui import run_app; run_app()

依赖：env 自带 jinja2/click/blinker/markupsafe 等大部分依赖；本
       包额外提供 flask 3.1.3 + werkzeug 3.1.8 + itsdangerous 2.2.0
       到 live-2d/_webui_vendor/，启动 bat 通过 PYTHONPATH 自动加载。
