# MemOS 快速启动指南 ⚡

## 🎯 三步开始使用

### 步骤 1️⃣：启动 MemOS API 服务

**双击运行**：`MEMOS-API.bat`

等待看到提示：
```
✅ MemOS 服务启动成功!
📍 向量存储路径: ./memos_data
🧠 Embedding 模型: ./RAG-model
```

> ⚠️ **重要**：不要关闭这个窗口！保持运行。

---

### 步骤 2️⃣：导入旧记忆（首次使用必须）

**方式A - 使用 WebUI（推荐）**：

1. 双击运行：`MEMOS-WebUI.bat`
2. 浏览器会自动打开管理界面
3. 选择"📥 导入记忆"页面
4. 点击"🚀 一键导入"按钮
5. 等待导入完成

**方式B - 使用迁移脚本**：

双击运行：`迁移记忆.bat`

---

### 步骤 3️⃣：启动肥牛AI

正常启动肥牛AI，现在它已经拥有记忆能力了！

## 🔧 日常使用

### 每次启动顺序：

1. 先启动：`MEMOS-API.bat` （保持运行）
2. 再启动：肥牛AI 主程序
3. 可选：`MEMOS-WebUI.bat` （管理记忆）

### 配置调整：

编辑 `live-2d/config.json`：

```json
{
  "memos": {
    "enabled": true,           // 开关
    "auto_inject": true,       // 自动注入
    "inject_top_k": 3         // 每次检索数量
  }
}
```

## ❓ 常见问题

**Q: 记忆不生效？**
- 检查 MEMOS-API.bat 是否运行
- 检查 config.json 中 `memos.enabled` 是否为 `true`

**Q: 如何关闭记忆功能？**
- 方式1：不启动 MEMOS-API.bat
- 方式2：修改 config.json 设置 `memos.enabled: false`

**Q: 记忆存储在哪里？**
- 向量数据：`./memos_data/`
- 原文件保留：`live-2d/AI记录室/记忆库.txt`
