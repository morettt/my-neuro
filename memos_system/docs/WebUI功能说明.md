# WebUI 功能说明

## 🎨 完整功能列表

### ➕ 添加记忆（两种模式）

#### 模式 1：🤖 智能加工（推荐）

**功能**：
- 输入原始对话或文本
- LLM 自动提取关键信息
- 自动去重合并（相似度>95%）
- 最大 2000 tokens 保留细节

**使用场景**：
```
输入: 我今天去踢足球了，感觉很开心，天气很好，认识了新朋友
加工: 用户今天踢足球感到开心；天气好；认识了新朋友
```

**操作**：
1. 选择"🤖 智能加工"模式
2. 输入对话内容
3. 点击"🤖 智能加工并保存"

#### 模式 2：✏️ 自由编写（直接存储）

**功能**：
- 直接编写精炼的记忆
- 不经过 LLM 加工
- 可设置重要度（0-1）
- 支持批量添加（1-10条）
- 自动去重检查

**使用场景**：
```
直接写: 用户生日是5月1日
重要度: 0.9
→ 直接存储
```

**操作**：
1. 选择"✏️ 自由编写"模式
2. 设置记忆条数
3. 输入每条记忆内容
4. 调整重要度滑块
5. 点击"💾 直接保存"

---

### 📋 记忆列表

**功能**：
- 查看所有记忆
- 显示时间戳和重要度
- ✏️ **编辑记忆**（新增！）
- 🗑️ 删除记忆

**编辑操作**：
1. 找到要编辑的记忆
2. 点击 "✏️" 按钮
3. 修改内容和重要度
4. 点击 "💾 保存" 或 "❌ 取消"

---

### 🔍 搜索记忆

**功能**：
- 语义搜索（不需要精确匹配）
- 显示相似度百分比
- 可调整返回数量（1-10）

**示例**：
```
搜索: 用户喜欢什么运动
结果: 用户喜欢踢足球 (相似度: 89%)
```

---

### 📥 导入记忆

**功能**：
- 一键导入旧记忆库
- 自动 LLM 加工
- 自动去重合并
- 批量加工现有记忆

**操作**：
1. **首次导入**：
   - 点击"🚀 一键导入"
   - 等待导入完成

2. **批量加工**：
   - 滚动到底部
   - 点击"🔧 开始加工"
   - 加工所有未处理的记忆

---

### ⚙️ 召回设置

**功能**：
- 快速预设（精准/平衡/宽松）
- 高级参数调整：
  - 返回数量
  - 相似度阈值
  - 时间衰减权重
  - 重排序开关

---

### 📊 统计

**功能**：
- 总记忆数
- 今日新增
- 本周新增
- 平均重要度
- 增长趋势图

## 🎯 使用流程

### 日常使用：

```
1. 添加记忆
   ├─ 对话内容 → 智能加工
   └─ 精炼内容 → 自由编写

2. 管理记忆
   ├─ 查看列表
   ├─ 编辑内容
   ├─ 调整重要度
   └─ 删除无用记忆

3. 搜索记忆
   └─ 语义搜索测试

4. 查看统计
   └─ 了解记忆增长情况
```

## 💡 最佳实践

### 添加记忆：

**智能加工模式**（推荐）：
- 适合：对话记录、长文本
- 优点：自动提取、去重合并
- 示例：粘贴整段对话

**自由编写模式**：
- 适合：关键信息、已整理的内容
- 优点：完全控制、快速添加
- 示例：用户生日；用户喜好等

### 重要度设置：

| 重要度 | 类型 | 示例 |
|-------|------|------|
| 0.9-1.0 | 核心信息 | 用户生日、姓名 |
| 0.7-0.8 | 重要偏好 | 喜好、习惯 |
| 0.5-0.6 | 一般信息 | 普通对话 |
| 0.0-0.4 | 临时信息 | 今天天气 |
