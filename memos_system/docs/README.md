# MemOS 记忆系统 - 使用指南

## 🚀 快速开始（3步）

### 步骤 1：启动 MemOS API
```
双击运行: MEMOS-API.bat
```
等待看到：
```
✅ MemOS 服务启动成功!
🧠 Embedding 模型: ./RAG-model
```

### 步骤 2：导入旧记忆（首次必须）
```
双击运行: 迁移记忆.bat
输入 y 确认
```

### 步骤 3：启动肥牛AI
```
正常启动肥牛
现在肥牛拥有记忆能力了！
```

## 🔧 配置调整

编辑 `live-2d/config.json`：

```json
{
  "memos": {
    "enabled": true,         // 总开关
    "auto_inject": true,     // 自动注入记忆
    "inject_top_k": 3       // 每次检索数量
  }
}
```

## ❓ 常见问题

**Q: 记忆不生效？**
- 检查 MEMOS-API.bat 是否在运行
- 运行 `测试MemOS.bat` 检查服务状态

**Q: 如何临时关闭？**
- 方式1：不启动 MEMOS-API.bat
- 方式2：config.json 设置 `memos.enabled: false`

**Q: 记忆存在哪？**
- `./memos_data/memory_store.json`

## 📊 技术细节

- **Embedding**: 本地 BGE 模型 (GPU加速)
- **存储**: JSON 文件
- **检索**: 余弦相似度
- **端口**: 8003 (API), 8501 (WebUI)
