## ❌ 常见问题排查

### 问题：WebUI 显示服务未启动
```
解决：
1. 检查 MEMOS-API.bat 是否在运行
2. 运行 测试MemOS.bat 检查服务
3. 检查端口 8003 是否被占用
```

### 问题：记忆召回不生效
```
解决：
1. 检查 config.json 中 memos.enabled = true
2. 检查控制台是否有"🧠 MemOS 检索"日志
3. 重启肥牛AI
```

### 问题：导入记忆失败
```
解决：
1. 检查文件路径是否正确
2. 检查 API 服务是否运行
3. 查看 MEMOS-API.bat 窗口的错误信息
```
