1. 创建并激活虚拟环境（不要忘了这一步！！！！第一步很重要！！）
```bash
conda create -n my-ai python=3.11 -y

conda activate my-ai
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3.打开ai

```bash
python main_chat.py

#或者也可以通过exe启动
python test.py

```
